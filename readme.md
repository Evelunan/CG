## 使用说明
### 1.选择节点
点击左侧“场景”，继续点击“Trans”，选择其中的“Geode”  
**注意：** 选择第一个“Geode” 无效，其他节点（线段、折线）均可

### 2.几何变换
选中节点后，点击平移面板中的“变换”按钮
**注意：** “变换”按钮只是放在了平移面板中，实际上是对选中节点进行几何变换的操作，不仅仅是平移
#### 2.1 平移
按住鼠标左键，移动鼠标即可实现图形的平移，松开鼠标结束移动
#### 2.2 旋转
按住鼠标右键，旋转鼠标即可实现图形绕基准点旋转，松开鼠标结束操作
#### 2.3 缩放
按住Ctrl键，滑动鼠标滚轮，图形将基于基准点缩放，向下滑动缩小，向上滑动放大
#### 2.4 选择基准点
点击鼠标左键并按住SHIF键，选中基准点
