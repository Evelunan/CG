## 实验3-使用说明
### 1.选择节点
点击左侧“场景”，继续点击“Trans”，选择其中的“Geode”  
**注意：** 选择第一个“Geode” 无效，其他节点（线段、折线）均可

### 2.几何变换
选中节点后，点击平移面板中的“变换”按钮  
**注意：** “变换”按钮只是放在了平移面板中，实际上是对选中节点进行几何变换的操作，不仅仅是平移
#### 2.1 平移
按住鼠标左键，移动鼠标即可实现图形的平移，松开鼠标结束移动
#### 2.2 旋转
按住鼠标右键，旋转鼠标即可实现图形绕基准点旋转，松开鼠标结束操作
#### 2.3 缩放
按住Ctrl键，滑动鼠标滚轮，图形将基于基准点缩放，向下滑动缩小，向上滑动放大
#### 2.4 选择基准点
点击鼠标左键并按住SHIF键，选中基准点

## 实验6-使用说明
**为了方便操作，实验6中将相机位置与被观察点之间的距离放大缩小的操作改为按住Ctrl后鼠标滚轮上下滚动**  
**为了观察方便，需要先切换至透视投影模式**
### 1、三维图形观察  
点击“相机控制”按钮进入

+：观察窗口放大到原1.25倍，-：（观察窗口缩小到原0.8倍）  
L：观察窗口左边界左移（减小），l: 观察窗口左边界右移（增加）  
R：观察窗口右边界左移（减小），r: 观察窗口右边界右移（增加）  
B：观察窗口下边界下移（减小），b: 观察窗口下边界上移（增加）  
T：观察窗口上边界下移（减小），t: 观察窗口上边界上移（增加）  
P：切换成透视投影  
O：切换成正交投影 

正交投影模式下按键实现：  
Ctrl+L：左视图，Ctrl+R：右视图；Ctrl+F：前视图；   
Ctrl+B：后视图；Ctrl+T：俯视图，Ctrl+D：顶视图（仰视图） 

相机旋转操作：  
按住鼠标左键上下移动：相机绕其X轴旋转   
按住鼠标左键左右移动：相机绕其Y轴旋转    

相机位置与被观察点之间距离放大缩小：  
**按住Ctrl后**，鼠标滚轮上下滚动
 
### 2、跟踪球模式  

点击“跟踪球模式”后进入    

相机旋转操作：  
按住鼠标左键上下移动：沿经线移动  
按住鼠标左键左右移动：沿维度线移动   

远离或靠近被观察点：  
**按住Ctrl后**，鼠标滚轮上下滚动
